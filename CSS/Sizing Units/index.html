vh and vw units

Viewport-based units vh and vw allow you to size elements relative to the browser's viewport dimensions:

  • vh (viewport height): 1vh is equal to 1% of the viewport's height. For example, if the viewport height is 900px, then 1vh = 9px.
  • vw (viewport width): 1vw equals 1% of the viewport's width. So if the viewport width is 1200px, then 1vw = 12px.
